# Satumenitaja

Project web berbasis Next.js.

## Development

```bash
npm install
npm run dev
```

Buka `http://localhost:3000`.

## Available scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Stack utama

- Next.js
- React
- TypeScript

## Deployment note

README ini tidak mencantumkan instruksi deployment spesifik platform. Konfigurasi production dan environment dikelola di luar repository.
